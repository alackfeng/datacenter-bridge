# datacenter-bridge (dcb)

## multiple datacenters bridge direct connection go library.
## 多区域数据中心建立直通桥, 进行跨区域收发数据.

## 功能设计
- [ ] 服务发现(ServiceDiscovery)
    - [ ] consul
    - [ ] etcd
    - [ ] redis
- [ ] 桥通信(BridgeChannel)
    - [ ] http(s)
    - [ ] websocket(s)
    - [ ] tcp(tls)
    - [ ] quic
    - [ ] p2p
    - [ ] mqtt
